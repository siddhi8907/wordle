WORDLE

Basically, you get 6 tries to find a secret 5-letter word. Every time you guess, the boxes change color to help you out. Green means the letter is perfect, yellow means it's in the word but wrong spot, and gray means its totally wrong.

I have used a package (random-words) to generate a 5 letter word and then each guess is checked letter by letter. we also have options to restart the game and to check the winning streak.

I learnt about how to implement the wordle logic, and keeping track of the green, yellow, used tiles. I also learnt how to implement a grid using html and css and all the layouts and spacing. I learnt how to link the html code with the javascript to make the project functional. I learned that small things, like the keyboard losing "focus" after clicking a button or the screen not fitting on a phone, can totally ruin a game.Handling "double letters" (so you don't get two yellow hints for one letter in the secret word) was a real brain-teaser.
