# wordle
## About Game:
You get 6 tries to find a secret 5-letter word.
* Green: Letter is perfect (right spot!).
* Yellow: Letter is in the word but wrong spot.
* Gray: Letter is just wrong.

## Tech Stack:
* HTML: for the basic structure
* CSS: for styling
* Javascript: for adding the logic

## Features:
* Real-time color-coded tiles (Green, Yellow, Gray) provide immediate heuristic feedback on every guess.
* Supports full keyboard interaction, including "Backspace" for corrections and "Enter" for submission, with built-in validation for 5-letter constraints.
* Automatic detection of game-over conditions with custom messaging for "Splendid" victories or revealing the word upon defeat.

## Problems Faced:
* Handling "double letters" (so you don't get two yellow hints for one letter) was a real brain-teaser.
* I learned how to fix bugs like the keyboard losing "focus" after clicking "New Game,".
* Fixing "Illegal return" errors taught me to keep my functions organized.
